# Library Management System

Desktop application developed with **Python + SQLite**.

## Features
- Full CRUD for books, members, and loan operations
- Advanced search and filtering
- Borrowing/return processes (stock updates with automatic triggers)
- Late payment tracking and reports
- User-friendly Tkinter interface

## Installation
```bash
git clone https://github.com/3bdullah3la2/library-management-system.git
cd library-management-system
python -m src.gui
